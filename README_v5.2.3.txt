SEUL-POLICE v5.2.3

이번 패치 내용만 포함합니다.

1. 근무자 근무순서 규칙 재정리
- A/B/C/D반 전체 적용
- 1발전/2발전 동일 규칙 적용
- 해당 월의 첫 근무일은 저장된 기준 순서 그대로 시작
- 다음 근무일부터 맨 뒤 근무자가 맨 앞으로 이동
- 휴무일은 회전 카운트에서 제외

예시:
7/1  1 2 3 4
7/2  4 1 2 3
7/3  3 4 1 2
7/4  휴무
7/5  2 3 4 1
7/6  1 2 3 4
7/7  4 1 2 3
7/8  휴무

2. 근무별순서 UI 정리
- N/A/D별 개별 순서 사용 중지
- 저장된 기준 근무자 순서 하나만 사용
- 근무지설정에서 수정/드래그/회전 가능

3. 직원DB 동기화 기준
- 직원DB는 Firebase employees 경로 기준으로 읽고 씀
- 휴대폰/PC가 같은 직원DB를 보도록 정리

4. 관리자설정
- 반별 직원목록은 기본 접힘 상태 유지
- 필요한 반만 펼쳐서 확인
