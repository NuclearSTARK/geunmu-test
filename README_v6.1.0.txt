SEUL-POLICE v6.1.0

이번 패치 내용만 기록합니다.

- ABD반 근무순서 엔진을 연속 회전 구조로 고정
- A/B/D반은 전체 근무일 기준으로 월/연도 변경 시에도 순서 유지
- C반은 A근무/D근무/N근무별 독립 회전 구조 유지
- 휴무일은 회전 카운트에서 제외
- 회전 방향을 1234 → 4123 → 3412 → 2341 방식으로 고정
- 근무자 이름과 순서 엔진을 분리
- 근무자는 월별로 변경 가능하지만 순서 로직은 유지
- 캐시 갱신용 버전을 6.1.0으로 업데이트

주의:
- 관리자설정, 순찰설정, 직원DB, 공지, 앱아이콘, UI 기능은 유지
- 근무순서 엔진 외 기능 변경 없음
