SEUL-POLICE v5.1.4-fix2

반영사항
1. 근무자 선택의 DB명단 적용 버튼 제거
2. 근무자 선택 후 '근무자 명단 저장' 버튼으로 확정 저장
3. 근무자 선택 변경 시 '변경됨' 표시
4. 공지사항 전광판 중복 표시 수정: Firebase 전역 공지 1개만 표시
5. 설정 탭에 '반별 고급설정' 복구
6. 설정 탭 순서: 개인설정 → 반별 고급설정 → 관리자설정
7. 개인설정 순서: 나의 반 → 이름 → 나의 근무지
8. 개인설정 이름 표시에서 '형대(형대)' 제거, 실명만 표시
9. 반별 고급설정 ON/OFF가 실제 근무지설정 화면에 반영됨

테스트 체크리스트
- 근무자 선택 후 저장 버튼을 눌러야 월간표에 반영되는지 확인
- 공지사항이 한 줄만 흐르는지 확인
- 다른 기기/시크릿창에서도 공지사항이 동일하게 보이는지 확인
- 설정에 반별 고급설정 탭이 보이는지 확인
- 근무지순서 사용 OFF 시 근무지순서 UI가 숨겨지는지 확인
- 근무별순서 사용 ON/OFF 시 해당 UI가 보이거나 숨겨지는지 확인
