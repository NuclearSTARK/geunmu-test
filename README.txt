SEUL-POLICE v5.0.5

업데이트 내용
1. 직원 DB 관리 화면 개선
   - 직원 DB가 A반 / B반 / C반 / D반 카드 형태로 나뉘어 표시됩니다.
   - DB에는 직원 이름과 반만 저장합니다.

2. 근무자 수 기본값 안정화
   - 기본 근무자 수는 4명입니다.
   - 직원 DB 명단 적용 시 직원이 많아도 자동으로 5명/6명으로 바뀌지 않습니다.
   - 근무자 수 5명/6명 변경은 관리자모드에서만 가능합니다.

3. 근무자 선택 방식 유지
   - 현재 선택한 반의 직원 DB만 표시됩니다.
   - 같은 월/반/발전 안에서 중복 선택이 방지됩니다.

4. 근무지(포지션) 선택 방식 변경
   - 기존 직접 타이핑 방식에서 선택 방식으로 변경했습니다.
   - 기본: 입초 / 기록 / 검색 / 소내
   - 5명/6명일 때: 출검 / 소내2 / 출모 / 소내3 후보도 표시됩니다.
   - 같은 월/반/발전 안에서 근무지 중복 선택이 방지됩니다.

5. 기존 기능 유지
   - 월별 독립 저장 유지
   - C반 N근무 순찰 표시: 이름🚔
   - Firebase 실시간 저장 유지
